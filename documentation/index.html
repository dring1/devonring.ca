HTML Structure

Whole template is based on Bootstrap 3 framework. Everything about this framework you will wind here.

Below you see the structure of the template:

<!-- Header  -->
<header id="nav-content" class="vertical-nav nav-white">

    << CONTENT GOES HERE >>

</header>
<!-- Header / End -->

<!-- Page Content -->
<div id="page-content">

    ...
    <!-- Section -->
    <section id="sections-name">

        << CONTENT GOES HERE >>

    </section>
    ...

</div>
<!-- Page Content / End -->

Every section is divided according to Bootstrap Grid System.

 

CSS Structure

Cured template uses five custom stylesheets:

<link rel="stylesheet" type="text/css" href="css/bootstrap.css" />
<link rel="stylesheet" type="text/css" href="css/style.css" />
<link rel="stylesheet" type="text/css" href="colors/blue.css" id="colors" />
<link rel="stylesheet" type="text/css" href="css/animations.css" />
<link rel="stylesheet" type="text/css" href="css/owl.theme.css">
  1. css/bootstrap.css (Bootstrap CSS file)
  2. css/style.css (main CSS file)
  3. color/color-name.css (sets template color)
  4. css/animations.css (css animations stylesheet)
  5. css/owl-theme.css (Owl carousel / slider theme)
 
and three defalut stylesheets:
 
<link rel="stylesheet" type="text/css" href="css/font-awesome.css">
<link rel="stylesheet" type="text/css" href="css/flaticon.css" />
<link rel="stylesheet" type="text/css" href="css/owl.carousel.css">

 

  1. css/font-awesome.min.css (Font Awesome stylesheet)
  2. css/flaticon.css (Font generated by flaticon.comunder CC BY. The authors are: Freepik.)
  3. css/owl.carousel.css (Owl Carousel stylesheet)

Bootstrap 3 icludes lots of useful components, which you may check here and here.
 
Cured template includes also some additional classes like:

.content - this class adds top padding
.black - use this class to section with black background
.dark  - use this class to section with dark background - it changes font color to white
.padded  - this class adds 60px padding
.padded-short  - this class adds 30px padding
.padded-vertical  - this class adds 70px top and bottom padding
.short - this class removes bottom margin
.medium - this class changes bottom margin to 10px
.long - this class changes bottom margin to 50px
